## 디렉토리 개요

Slack 이벤트를 받아 Claude Agent SDK를 실행하고, Slack 메시지로 결과를 스트리밍/게시하는 에이전트 런타임 계층이다.

## 코드 작성 컨벤션

- Slack 쓰레드 기준으로 세션을 식별하고, Claude Agent SDK의 `resume`를 이용해 멀티턴을 유지한다.
- 진행 상태는 메시지 마지막에 `:loading-dots: {{name}}이(가) 생각 중이에요` context block을 먼저 표시하고, 스트리밍 응답은 그 위에 prepend하여 업데이트한 뒤 최종 응답 시 context를 제거한다. 블록 출력이 Slack 제한을 초과하면 새 메시지를 만들어 이어서 업데이트하고, 이때 기존 메시지의 생각 중이에요 context는 제거한다. 메시지는 각 턴마다 새로 만든 뒤 에이전트 출력(비결정적 내용)만 업데이트한다.
- Slack 출력은 코드 블록/인라인 코드를 제외하고 Markdown을 Slack mrkdwn에 맞게 정규화해서 전송한다.
- Slack API 호출 실패 시에도 사용자가 원인을 이해할 수 있도록 최소한의 안내 메시지를 남긴다.
